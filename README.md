# YuwanaDev MDM - Dashboard

The administrative web interface for the YuwanaDev Mobile Device Management (MDM) platform. It allows administrators to monitor device fleets, execute remote commands, and view real-time telemetry.

## Features
- **Device Inventory**: Real-time monitoring of all connected Android assets.
- **Live Telemetry**: View battery, temperature, storage, and active networks.
- **Remote Commands**: Ping, lock, take screenshots, and broadcast alerts.
- **Screen Mirroring**: Real-time remote screen view using WebRTC.
- **Minimalist UI**: Clean, responsive, and modern interface.

## Tech Stack
- **Framework**: Next.js 14 (App Router)
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **State Management**: Zustand
- **Icons**: Lucide React

## Build & Run
To run the dashboard locally:
```bash
# Install dependencies
pnpm install

# Start the development server
pnpm dev
```
Open [http://localhost:3000](http://localhost:3000) in your browser.
